This fork gives you a clear relationship of each part in the project corundum. Each part are separated.

src_driver is the path for all the driver files.

src_Vivado is the path for Vivado. You can simply open the project in through your Vivado.

xdc_files is the path for constraints, including pin constraints and timing constraints. config.tcl also needs to be applied before implementation.

src_userspace_tool is the path for all the tools

src_verilog is the path for verilog source files.
